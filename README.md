## Docker環境セットアップ手順

### Set Up
Docker for Macをインストールする

[Docker for Mac website](https://docs.docker.com/docker-for-mac/install/#download-docker-for-mac)


```

## repositoryをclone
git clone git@github.com:ebarakazuhiro/docker-lumen.git

## docker images作成
docker-compose build

## docker 起動
docker-compose up


```
